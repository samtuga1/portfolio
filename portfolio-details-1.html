Details of MyShop app

Project information

  • Category: Mobile development
  • Project date: 19 April, 2022

I used firebase authentification to authenticate users.I also used firebase realtime storage to store all my necessary data such as products, orders and more. I also used Sharedpreferences package to store the user token, User ID and expiry time firebase gives every user so that the user can automatically login the app
